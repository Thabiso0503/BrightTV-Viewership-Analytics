# 🎬 BrightTV Viewership Analytics & Growth Strategy

## 📊 Project Overview
This project presents a data-driven analysis of BrightTV’s user behavior and viewing patterns. The goal is to generate actionable insights that support the **Customer Value Management (CVM)** team in achieving the company’s objective of growing its subscription base for the financial year.

The analysis focuses on understanding **who the users are, how they consume content, and what drives engagement**, ultimately leading to strategic recommendations.

---

## 🎯 Business Objective
BrightTV’s CEO aims to:
- Increase the **subscription base**
- Improve **user engagement and retention**

This project provides insights to help:
- Identify viewing trends  
- Understand consumption drivers  
- Recommend content strategies  
- Highlight growth opportunities  

---

## 📁 Dataset Description
The dataset includes:
- **User Profiles**
  - Demographics (e.g., age group)
- **Viewer Transactions**
  - Session times
  - Viewing duration
  - Content consumed  

---

## 🔍 Key Analysis Areas

### 1. User & Usage Trends
- Majority of users are **youth (18–35)**
- Peak consumption occurs during:
  - **Evenings (18:00 – 22:00)**
  - **Weekends**

👉 Insight: BrightTV is primarily a **youth-focused entertainment platform**

---

### 2. Factors Influencing Consumption
Key drivers identified:
- Time availability (after work/school)
- Content relevance (entertainment-driven)
- Accessibility (mobile-friendly)
- Trending/popular content

👉 Insight: Users prefer **engaging, relatable, and easily accessible content**

---

### 3. Low Consumption Patterns
- Lower activity observed:
  - **Weekdays (Mon–Wed)**
  - **Morning hours**

👉 Insight: Users lack strong incentives to engage during busy periods

---

### 4. Content Strategy Opportunities
To increase engagement during low periods:
- Introduce **short-form content**
- Release **daily episodes or mini-series**
- Use **push notifications**
- Add **interactive features** (polls, live sessions)

---

## 🚀 Recommendations

### 📺 Content Strategy
- Invest in **youth-oriented and trending content**
- Focus on **entertainment and relatable storytelling**

### 📈 Engagement Strategy
- Implement **personalized recommendations**
- Use **notifications during peak hours**

### 📅 Weekday Activation
- Launch **weekday-exclusive content**
- Introduce **gamification (rewards, streaks)**

### 📢 Marketing Strategy
- Target youth through **social media campaigns**
- Collaborate with **influencers**

### 🔁 Retention Strategy
- Improve user experience
- Offer **subscription bundles and promotions**

---

## 💡 Key Takeaway
BrightTV’s growth depends on:
- Understanding **viewer behavior**
- Delivering **relevant content**
- Engaging users at the **right time**
